NV

Пакетный менеджер, через который можно ставить и обновлять приложения по схеме @creator/package. Сначала ставишь NV, потом работаешь одной короткой командой.

Скачать NV Открыть каталог
creator/package latest по умолчанию 
# проверить версию менеджера
nv -v

# поставить пакет
nv install @lvls/neuralv

# поставить конкретную версию
nv install @lvls/neuralv@1.5.0

Ставит

Берёт пакет из каталога и ставит его на нужную систему.

Обновляет

Повторная команда установки тянет актуальную версию пакета.

Показывает каталог
